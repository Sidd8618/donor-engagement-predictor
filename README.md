# Donor Engagement Prediction for NayePankh Foundation

# Overview

This project develops a Machine Learning model to predict donor engagement for future campaigns. The goal is to help NayePankh Foundation identify supporters who are most likely to respond to fundraising and awareness initiatives, enabling more effective and data-driven outreach.

## Dataset

Dataset: marketing_campaign.csv
Source: Kaggle
Features: Income, Recency, Purchase Behavior, Website Activity, Campaign Responses, and Demographics

# Machine Learning Models

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier

# Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

# Results

The Random Forest model achieved the best overall performance and was selected as the final model .
